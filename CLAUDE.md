## Agent skills

### Issue tracker

Issues live in GitHub Issues at `github.com/rboudrouss/astro-cms/issues`, managed via the `gh` CLI. See `docs/agents/issue-tracker.md`.

### Triage labels

Standard label vocabulary: `needs-triage`, `needs-info`, `ready-for-agent`, `ready-for-human`, `wontfix`. See `docs/agents/triage-labels.md`.

### Domain docs

Single-context repo. `CONTEXT.md` at the root, ADRs in `docs/adr/`. See `docs/agents/domain.md`.
